## Il Plurale delle parole in -ca/-ga & -co/-go-regola

Al femminile sempre:
* -ca -> -che
* -ga -> -ghe

Betonung auf der drittletzten Silbe:
* -co -> -ci
* -go -> -gi

Betonung auf der vorletzten Silbe:
* -co -> -chi
* -go -> -ghi

* di solito - usually
* qualche volta - sometimes
* a volte - sometimes
* sempre - always

* mai - never

Nach dem Verb:
* Mangio sempre a casa

Negativ (vor dem Verb):
* Non bevo mai alcolici
